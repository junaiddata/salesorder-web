Bulk Import PDFs

Upload a ZIP file to import datasheets, test certificates, and other PDFs for materials.

ZIP structure

BRAND_CODE/
  model_no/
    datasheet/ or datasheets/  → technical_pdf (Section 10)
      filename.pdf
    country_of_origin/   → country of origin certs (multiple)
      cert1.pdf
    previous approvals/  → previous approvals (multiple)
      approval.pdf
    test certificates/   → test certs (multiple)
      cert1.pdf
      cert2.pdf
    catalogue/           → catalogue_pdf (Section 9) — optional
      catalogue.pdf

Brand code must match exactly (e.g. PEGLER). Model numbers must exist in Materials.
